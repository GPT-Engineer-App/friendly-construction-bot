# friendly-construction-bot

a generic website for a local construction company, specializing in everything from the slab up. it should have a chatbot in the corner that can offer to book a time or request more information. it should also have a modern look and flow easily

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/friendly-construction-bot.git
cd friendly-construction-bot
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
